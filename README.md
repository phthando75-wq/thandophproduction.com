## 🚀 Quick Start

### Option 1: GitHub Pages (Recommended)
1. Go to your repository settings
2. Navigate to **Pages** section
3. Select `main` branch as source
4. Your site will be live at: `https://phthando75-wq.github.io/thandophproduction.com`

### Option 2: Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/phthando75-wq/thandophproduction.com.git
   cd thandophproduction.com
   ```
2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```

## 🎨 Customization

### Update Your Image
Edit `index.html` line 51 and replace the placeholder URL:
```html
<img src="YOUR-IMAGE-URL-HERE" alt="DJy Thando PH live on the decks" class="hero-img">
```

### Update Social Media Links
Find the social links section (around line 103) and update URLs:
```html
<a href="https://spotify.com/..." class="social-btn">Spotify</a>
```

### Change Color Scheme
Edit `styles.css` lines 1-5 to customize colors:
```css
:root {
    --primary-color: #111;
    --accent-color: #ff8c00;  /* Change this to your color */
}
```

### Update Contact Information
Modify the contact form in `index.html` to connect with your email service (Formspree, Netlify, etc.)
